# **App Name**: AI Website Crafter

## Core Features:

- AI-Powered Website Generation: Generates a basic website structure and content based on user input. This tool will decide what content makes sense to include, if any. Leverages the Hugging Face API.
- Template Selection: Allows users to choose from a selection of pre-designed website templates to be customized.
- Customization Options: Provides options to customize the generated website's text, images, and basic layout through a visual editor.
- Preview Mode: Enables users to preview the website before deploying or downloading.
- Simple Deployment: Offers a one-click deployment option to a static hosting provider like Netlify or Vercel (manual setup).
- Admin interface: Simple interface for administrative tasks
- Authentication: Allow users to create a basic username and password

## Style Guidelines:

- Primary color: Soft Lavender (#D0BFFF) to convey creativity and calmness.
- Background color: Light Gray (#F5F5F5) to ensure content readability and a clean interface.
- Accent color: Muted Rose (#E0AED6) for interactive elements and highlights.
- Headline font: 'Poppins' (sans-serif) for a modern, geometric, and approachable feel.
- Body font: 'PT Sans' (sans-serif) to maintain readability and complement the headline font. 
- Use simple, line-based icons that align with the overall clean and modern aesthetic.
- Maintain a clean and intuitive layout with clear sections for different functionalities.
